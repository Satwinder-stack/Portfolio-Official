Coffee Cat

PURPOSE

Creating a digital home for Coffee Cat Cafe, a hidden cafe that features cats as its main attraction. The project focuses on introducing and enhancing the cafe's popularity in the internet, this also allows feline enthusiasts to meet the adorable cats, view cat profiles, and explore a specialized menu in a relaxing environment.

CORE FEATURES

• Interactive cat profile gallery with personalized personalities.
• Warm aesthetics and UI specialy made for this cafe.
• Digital cafe menu with dietary filters and seasonal specials.
• Quick introductory to how the cafe started and its history.

TECHNOLOGIES USED

HTML / CSSCalming Layout & Warm Aesthetics
JavaScriptSmooth Transitions & On-Page Search Engine
JSONEfficient Storage container for lists of menu items
InfinityFree / FileZillaOnline Deployment
